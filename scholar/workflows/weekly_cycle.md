# Workflow — Weekly Cycle

- Role: The Scholar
- Objective: Maintain system health and evolution.

## Monday: Audit

- Audit the 3 most recent session logs.
- Update `scholar/outputs/reports/synthesis_recent_logs.md`.

## Wednesday: Deep-Dive

- Select 1 module dossier to update/create.
- Conduct web research; add 2+ notes to `scholar/outputs/research_notebook/`.

## Friday: Promotion

- Review the Promotion Queue.
- Finalize at most 1–2 RFCs for user review.
- Update `scholar/outputs/system_map/system_map.md` if needed.
