3. ADDED STALE ID FILTERING EFFECT (Lines 245-257)
   NEW useEffect that:
   - Runs whenever chatMaterials changes (fetched from API)
   - Creates Set of valid material IDs from API response
   - Filters selectedMaterials to only include valid IDs
   - Silently discards stale/deleted material IDs
   - Only updates state if something was filtered (prevents unnecessary re-renders)
   
   Code:
   ```typescript
   useEffect(() => {
     if (chatMaterials.length === 0) return;
     const validIds = new Set(chatMaterials.map((m) => m.id));
     setSelectedMaterials((prev) => {
       const filtered = prev.filter((id) => validIds.has(id));
       if (filtered.length < prev.length) {
         return filtered;
       }
       return prev;
     });
   }, [chatMaterials]);
   ```
