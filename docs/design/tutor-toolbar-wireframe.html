Pipeline reads left-to-right. Workbench (Workspace, Notes) and Settings (Run Config, Memory) are pinned to the right — reach them from any stage.

What flows where

LOAD
Source Shelf → pulls materials into the run
READ
Document Dock → view materials and take inline notes
PRIME
Priming chat runs prime methods on the material → output drops into Prime Packet
TUTOR
Tutor chat consumes Prime Packet for the live session → output drops into Polish Packet
POLISH
Polish chat refines what the tutor produced
EXPORT
Obsidian writes back to the vault. Anki stages cards.
WORKBENCH
Workspace collects material from any stage for synthesis. Notes is a freeform scratch surface (multi-instance).
SETTINGS
Run Config shapes Priming + Tutor. Memory resumes the next session.
