# Skills Catalog

Purpose: full per-skill catalog with descriptions and decision fields so the skill set can be reviewed one by one over time.

Decision values:
- `keep`
- `archive`
- `customize`
- `merge`
- `review-later`

Current review focus: `planning`

Seeded planning recommendations:
- keep as daily core: `plan-work`, `planner`, `prd`
- keep if you actively use Conductor planning: `conductor-new-track`
- merge/customize candidates: `ask-questions-if-underspecified`, `requirements-clarity`
- niche deep-planning options: `gepetto`, `llm-council`
- move out of planning core: `conductor-implement`
- weak/overlapping validator candidate: `conductor-validator`

Review workflow:
1. pick one category
2. go skill by skill
3. record `Decision` and `Notes`
4. only archive/delete after a deliberate pass
