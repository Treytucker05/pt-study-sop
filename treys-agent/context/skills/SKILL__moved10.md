# OpenAI Agent Builder (beginner)

## What it is
- A visual canvas to build multi-step agent workflows with nodes and typed connections.

## Rookie steps
1) Open Agent Builder in the OpenAI platform.
2) Start from a template or a blank workflow.
3) Add nodes, configure inputs/outputs, and connect edges.
4) Use Preview to test with sample inputs.
5) Publish to create a versioned workflow ID.
6) Deploy via ChatKit (embed) or export code for the SDK.

## When to choose
- You want a no-code or low-code workflow builder.
- You want fast iteration with visual debugging.
