# UI Images

Canonical repo-local home for shell and navbar image assets.
These images support the live CP-MSS v1.0 / Control Plane dashboard shell and Tutor-facing navigation surfaces.

## Layout

- root: live app images imported through `@assets`
- `buttons_for_dashboard_split/`: split source PNGs plus placement metadata for the custom navbar
- `reference/`: older shell art, previews, and working/reference images kept for iteration
- `generated_images/`: generated concept art kept out of the live import path
