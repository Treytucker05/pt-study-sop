# Specification: Agent Ecosystem Hygiene

**Track ID:** agent-ecosystem-hygiene_20260313
**Type:** Infra
**Created:** 2026-03-13
**Status:** Draft

## Summary

Normalize the multi-CLI agent and skill ecosystem so the supported tools project the same canonical shared-skill topology, broken active roots are repaired or explicitly quarantined, repo/operator docs reflect the real machine state, and supported active config no longer stores plaintext secrets.

## Context

The current machine-level agent ecosystem is only partially aligned:

- `C:\Users\treyt\.agents\skills` is the canonical shared-skill upstream.
- Claude, Codex, and Cursor project that upstream cleanly.
- Gemini and Antigravity currently carry broken active skill junctions.
- Cursor is omitted from the shared-skill README even though it consumes the shared upstream.
- `scripts/sync_agent_skills.ps1` does not match the real consumer matrix.
- `C:\Users\treyt\.gemini\antigravity\mcp_config.json` stores plaintext credentials inline and needs hardening.

This track turns the audit findings into a verified repair plan without guessing at unsupported roots or deleting shared upstream skills blindly.

## Acceptance Criteria

- Supported active skill roots are frozen in a documented topology matrix and have zero broken junctions after repair.
- `scripts/sync_agent_skills.ps1` matches the supported topology and passes `DryRun`, `Apply`, `Check`, and idempotence verification.
- Rollback artifacts exist before any home-directory mutation.
- Supported active config contains no plaintext credentials; any remaining rotation work is explicitly recorded with operator evidence or blocker status.
- Repo/operator docs reflect the real consumer matrix, Windows paths, local exceptions, and validation workflow.

## Non-goals

- Blind pruning of shared skills from `C:\Users\treyt\.agents\skills`.
- Converting config-only tools like Kimi or Conduit into shared-skill consumers without runtime proof.
- Broad plugin/version upgrades across the installed CLIs.
- Cleaning historical backups/transcripts unless a supported active tool still consumes them.
