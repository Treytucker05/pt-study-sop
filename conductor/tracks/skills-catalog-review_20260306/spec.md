# Specification: Skills Catalog Review

**Track ID:** skills-catalog-review_20260306
**Type:** Docs
**Created:** 2026-03-06
**Status:** Draft

## Summary

Create a full skill catalog file that lists every shared skill and tool-local exception with a short description and a decision column so skills can be reviewed one by one over time.

## Acceptance Criteria

- A repo file exists with one row per skill.
- Each row includes at least:
  - skill name
  - folder/source
  - whether it is shared or tool-local
  - short description
  - review bucket
  - decision field
  - notes field
- The catalog is practical for iterative review across future turns.
