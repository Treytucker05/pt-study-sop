# Specification: Tutor 10/10 Certification

**Track ID:** tutor-10-certification_20260307  
**Type:** certification  
**Created:** 2026-03-07  
**Status:** Active

## Summary

Define 10/10 as a hard release gate for the Tutor. The Tutor only counts as 10/10 when all selectable template chains pass certification, common study materials load and scope correctly, notes always save correctly to Brain and Obsidian, card-producing chains always create correct drafts, and the three neuro golden paths pass live end-to-end.

## Release Bar

- Zero Sev-1 failures anywhere in the certified surface
- Zero Sev-2 failures on neuro golden paths
- All P0/P1 scenarios green
- No selectable template chain left uncategorized or uncertified

## Supported 10/10 Surface

- Material scope:
  - PDF
  - PPTX
  - DOCX
  - TXT/MD
  - MP4
  - folder sync
  - selected-material scoping
- Chain scope:
  - all selectable user-facing template chains
- Output scope:
  - notes always save to Brain + Obsidian
  - card-producing chains always create correct drafts
  - final Anki push remains chain-conditional and is not part of the baseline release bar

## Initial Deliverables

- scorecard and severity policy
- template-chain certification registry
- deterministic fixture plan
- milestone-by-milestone certification matrices
- executable certification runner/report artifact
