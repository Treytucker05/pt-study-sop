PT Study Navbar Layout

Select the dashboard shell background and the eight split button PNGs. The plugin will build a 1536 x 1024 navbar frame in the current design file.

1. Background

Use C:\pt-study-sop\UI Images\Dashboard finished.png.

2. Button PNGs

Select all eight files from C:\pt-study-sop\UI Images\buttons_for_dashboard_split.

Expected button names

  • brain.png
  • scholar.png
  • tutor.png
  • library.png
  • mastery.png
  • calendar.png
  • methods.png
  • vault.png
Select the files and then click Build Navbar Frame.
